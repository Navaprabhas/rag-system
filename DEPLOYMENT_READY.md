## 🐛 Troubleshooting

### If build fails:

1. **Check file names:**
   - Must be exactly: `app.py`, `requirements.txt`, `README.md`
   - Case-sensitive!

2. **Check logs:**
   - Go to Space → "Logs" tab
   - Look for error messages

3. **Re-upload files:**
   - Delete old files
   - Upload fresh copies from `huggingface_deployment/`

### If Space shows "Building...":

- **This is normal!** First build takes 2-3 minutes
- Check "Logs" tab for progress
- Wait patiently

### If app doesn't load:

1. Refresh browser
2. Clear cache
3. Check logs for errors
4. Restart Space (Settings → Restart)

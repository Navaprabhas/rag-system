# 🔍 Production-Grade RAG System

[![GitHub](https://img.shields.io/badge/GitHub-Navaprabhas%2Frag--system-blue?logo=github)](https://github.com/Navaprabhas/rag-system)
[![Python](https://img.shields.io/badge/Python-3.11+-blue?logo=python)](https://www.python.org/)
[![FastAPI](https://img.shields.io/badge/FastAPI-0.109-009688?logo=fastapi)](https://fastapi.tiangolo.com/)

A complete, production-ready Retrieval-Augmented Generation (RAG) system with strict anti-hallucination guarantees.

## ✨ Features

- **📄 Document Ingestion**: PDF, TXT, and URL parsing
- **🔍 Advanced Retrieval**: Query rewriting, dense retrieval, cross-encoder re-ranking
- **🤖 Multi-Provider LLM**: Ollama (local), OpenAI, Anthropic
- **🛡️ Anti-Hallucination**: 4-layer validation system
- **⚡ Streaming Responses**: Real-time answer generation
- **📚 Citation Enforcement**: Every answer includes sources

## 🚀 Quick Start

```bash
git clone https://github.com/Navaprabhas/rag-system.git
cd rag-system
make setup && make start
```
